# Sali-Chat
一个多人在线聊天的node app<br>
项目上线地址:http://199.247.29.196:8080

## 多人同时交流
你可以使用Sali-chat发送消息，你发送的消息将会让任何一个之前曾经登录过得用户下次登录时看见。

## 消息实时记录
您的消息会被忠实的记录并保留，并保证可以传递给每一个登录的用户，不会有丢失和删除。

## 丰富的细节体验
您可以回车发送消息，消息列表可以支持自动滚动，你可以随时查看在线的人们。<br>
服务端基于node开发，高效处理您的消息。

## 开放的聊天环境
聊天室目前可以支持任意用户登录。登录凭据仅为用户名。新用户可以随时注册。

## 安全通信
基于WebSocket技术，提供安全的消息通信体验。您的聊天只会在数据库形成副本，不会公开。

## 发展方向
未来会开放API接口，点对点私密通信，可以设置消息加密，可以跨平台支持，一切只为了您的使用体验。
